# LibraryManagementSpringLab

This project is now set up as a standard Maven-based Spring Boot application with Maven wrapper files.

## Structure

- src/main/java/com/example/librarylab/DemoApplication.java
- src/main/java/com/example/librarylab/controller/
- src/main/java/com/example/librarylab/service/
- src/main/java/com/example/librarylab/repository/
- src/main/java/com/example/librarylab/model/
- src/main/java/com/example/librarylab/entity/
- src/main/resources/application.properties
- pom.xml
- mvnw
- mvnw.cmd
- .mvn/wrapper/

## Run

From PowerShell:

```powershell
cd "e:\Cognizant_DN_Deepskilling\Module 5\LibraryManagementSpringLab"
./mvnw.cmd spring-boot:run
```

## Test

```powershell
cd "e:\Cognizant_DN_Deepskilling\Module 5\LibraryManagementSpringLab"
./mvnw.cmd test
```
